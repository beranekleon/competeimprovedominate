# CID - Compete. Improve. Dominate

## Project Overview
This repository contains the complete codebase for our practise diploma project in a monorepo.

## The Goal
We aim to create an app that makes it easy for users to stay active and exercise by:
* **Conquerring territories:** users can claim real-world areas based on their location data
* **Competing with eachother:** users can claim other users territories, creating a fun and competitive game

## Tech Stack
To ensure performance and scalability our project is built with:
* **React Native**, creating a cross-platform mobile app
* **Node.js**, deployed in the Google Cloud
* **Firestore**, for real-time data synchronization
* **CI/CD**, ensuring automated deployment in the google cloud with every change in the codebase
